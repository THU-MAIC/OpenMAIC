# OpenMAIC - Project Overview

**OpenMAIC** (Open Multi-Agent Interactive Classroom) is an open-source AI platform that turns any topic or document into a rich, interactive classroom experience.

## Purpose
Powered by multi-agent orchestration, it generates slides, quizzes, interactive simulations, and project-based learning activities — all delivered by AI teachers and AI classmates who can speak, draw on a whiteboard, and engage in real-time discussions.

## Key Features
- One-click lesson generation from topics or uploaded materials
- Multi-agent classroom with AI teachers and peers
- Rich scene types: slides, quizzes, interactive HTML simulations, PBL (project-based learning)
- Whiteboard & TTS (text-to-speech) support
- Export to `.pptx` slides or `.html` pages
- OpenClaw integration for chat app access (Feishu, Slack, Telegram, etc.)
